## 简易待办事项 （PySide6 + QML）

一个基于 `PySide6 + QML` 的桌面事务管理工具，强调低干扰、卡片式操作和时间感知。

适合记录带开始时间、截止时间、提醒、标签和资料附件的日常事务。

## 主要特性

- 卡片式事务管理：进行中 / 已完成两类视图
- 内联编辑：双击卡片直接修改内容
- 时间轴可视化：开始、截止、当前时间一目了然
- 手势操作：
  - 左右滑动切换完成状态
  - 长按拖拽排序
- 紧急标记：卡片顶部 `!` 按钮，带强调视效
- 标签系统：
  - 标签胶囊展示
  - 多选标签筛选
  - 一键清空筛选
- 提醒通知：
  - 支持提前若干分钟提醒
  - 本地桌面通知
- 逾期状态：
  - 卡片斜线遮罩提示
  - 顶部统计动态更新
- 资料留存：
  - 支持保存网页链接
  - 支持导入本地文件
  - 支持拖拽文件到资料区添加
  - 资料目录可自定义
- 窗口增强：
  - 深浅主题切换
  - 窗口置顶
  - 贴顶微缩预览

## 运行环境

- Python `3.10+`
- Windows `10/11`

## 快速开始

### 安装依赖

```bash
python -m pip install -r requirements.txt
```

### 启动应用

```bash
python -m src.app
```

## 使用说明

### 1. 新建事务

- 点击顶部中间的 `+`
- 新事件会自动出现在列表底部，并进入可编辑状态

### 2. 编辑事务

- 双击卡片进入编辑
- 使用右下角 `✓` 保存
- 使用右下角 `×` 取消

可编辑内容包括：

- 标题
- 备注
- 标签
- 开始时间 / 截止时间
- 提醒时间
- 资料链接与文件

### 3. 卡片交互

- 左右滑动卡片：切换完成 / 恢复
- 长按卡片：拖拽排序
- 点击顶部 `!`：切换紧急状态
- 点击右侧删除图标：删除当前事务

### 4. 时间轴

- 卡片会显示开始时间、截止时间和当前时间的相对位置
- 编辑态下可直接拖动时间轴标记
- 拖动时按天吸附

### 5. 标签

- 标签以胶囊形式显示在卡片上
- 编辑时可直接输入逗号分隔标签
- 也可以在编辑态快速添加 / 删除标签
- 点击卡片中的标签可参与顶部筛选

顶部标签栏支持：

- 多选筛选
- OR 逻辑匹配
- 一键清空

### 6. 提醒

- 可设置提醒时间为：
  - `5 / 10 / 15 / 30 / 60` 分钟
  - 或手动输入分钟数
- 提醒基于：

```text
截止时间 - 提前N分钟
```

通知文案：

- 截止前：事件即将逾期
- 截止后：`<标题> 已逾期。`

说明：

- 这是应用运行时的本地通知
- 需要系统通知权限可用

### 7. 资料留存

每张卡片都可以挂资料，适合保留网页、文档、截图、附件等参考信息。

支持两类资料：

- 网页链接
- 本地文件

#### 添加网页链接

- 在编辑态资料区输入链接后回车
- 或点击 `读取剪贴板链接`

如果输入的是普通域名，应用会自动补成 `https://...`

#### 添加文件

- 点击 `导入文件`
- 或直接把文件拖到资料区的拖拽区域

#### 编辑资料

- 编辑态下，点击资料项本体即可移除

#### 打开资料

- 查看态下，点击资料项或右侧打开按钮即可打开

#### 修改资料存放目录

- 在编辑态资料区点击 `更改目录`
- 选择新的资料目录后，后续导入文件将保存到新位置

说明：

- 修改目录只影响之后新导入的文件
- 已经导入的旧文件不会自动搬迁

## 数据存储

### 事件数据

- 开发模式：

`data/events.json`

- 打包后：

`%APPDATA%\事务时间表项目v2\events.json`

### 资料目录

默认情况下：

- 开发模式：

`data/materials/`

- 打包后：

`%APPDATA%\事务时间表项目v2\materials\`

如果你手动修改了资料目录，则会使用你选择的新路径。

### 配置文件

当前会保存资料目录等设置：

- 开发模式：

`data/settings.json`

- 打包后：

`%APPDATA%\事务时间表项目v2\settings.json`

## 项目结构

- `src/app.py`：应用入口
- `src/storage.py`：本地数据与配置读写
- `src/viewmodels.py`：仓库、列表模型、资料导入与打开逻辑
- `src/reminders.py`：本地提醒服务
- `src/qml/Main.qml`：主窗口与页面结构
- `src/qml/EventCard.qml`：事务卡片、编辑态、资料区
- `src/qml/TimelineBar.qml`：时间轴组件
- `run_app.py`：打包入口

## 常见说明

- 贴顶微缩属于增强交互，在多屏或不同缩放比例下可能会有轻微行为差异
- 桌面通知是否显示受系统通知设置影响
- 资料目录修改后，旧资料不会自动迁移
